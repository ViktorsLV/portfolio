Web application

During the start of 2nd semester I had an opportunity to build a web application - a one page website, where content is generated through JavaScript

The goal was to build an application running in the browser with a feel of native application.

For our client - Danish Diabetes Association we wanted to create a "functional prototype" to a possible application which people with Diabetes type 2 could use to simplify their shopping needs and other concerns

Before the making of the actual website many steps before were done:

  • - GitHub repository set-up
  • - User Research
  • - Collection of data
  • - Low and high fidelity mock-ups
  • - Creation of Personas
  • - Creation of Icons
  • - Structuring of content (Information Architecture)

My roles in the project:
  • - User research (interview collection and creation of Personas)
  • - Frontend Developer
Technologies and tools used in the project:
  • HTML5
  • CSS3
  • JavaScript
  • Adobe XD
  • Adobe Illustrator
  • Trello
  • GitHub
