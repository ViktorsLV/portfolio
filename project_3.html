Dynamic webpage

Project in which some parts of the website had to be generated dynamically

The clients wish was to display the upcoming events in an understandable manner and display stories from past events as well as a good introduction to other offers from Blokhus Sport & Events

In order to complete the task the project group decided to use a Content Management System (CMS) - WordPress. The project was made from scratch during the period of 4 weeks, including these steps:

  • - Thorough project planning
  • - Collection of data (interviews)
  • - Creation of Personas
  • - Card sorting, content creation
  • - Low and high fidelity mock-ups
  • - Set up of CMS systems
  • - Testing
  • - Structuring of content (Information Architecture)
  • - SEO optimization

My roles in the project:
  • - Creation of Personas
  • - UI Designer
  • - Part of testing process
  • - CMS set-up
  • - Frontend Developer
